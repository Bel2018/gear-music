Software/Daw

Software is also called as DAW is another important and useful tools in music industry. You have to install the software in computer and run it when you record the tracks. Software, there are several kind of DAW you can use.

I would like to share a little opinion about the software. There many different kind of software programs out there. The names are just different, but I believe they all might be similar to use. I have not used all of them, so I am not sure how much they are different and similar exectly. But, my point here is to not buy a different software if you already have one because you think they might be different and might give you magical ways to use. I believe you are better to get used to with whatever you have by trying to work with it. Of course one day, you may want to upgrade your DAW to a more expensive program which has more plug in and tools to use. But, you can always start whatever program to build up your skill.
