Mixer

Do you really need to have mixer in studio? It would be best if you have one, but in my opinion, you don't need to have one, unless you really have a big studio and all your band mates for the recording the same time. Mixer has it own effects, so it would be easier to use mixer instead mixing in computer. It depends on you if you have big budget. As long as you have the interface and plug in program install, it mostly come with the software, you would be fine.
