Computers

Computer is one of the most needing in having a music studio. I would say, it is the main equipment that you need to start with. Here is thing, don't over worry about getting a fancy computer. When I start searching for studio computer, I was thinking only imac computer worked because I only had search musicians' studion that has imac computer. In reality, any kind of computer would work as long as they still run well.


So, computer is really need in order to have home studio, but it has doesn't have to be an expensive or fancy computer. You can buy any kind of confortable computer that fit your budget. Of course you may one day one day would have a nice computer to work with, but you don't need to have an expensive computer to begin with.
